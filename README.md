is_maybe - 0.0.5
====================

Usage
=====

- `npm i -D is_maybe`

    or

- `<script src="./is_maybe.js"></script>`


Example use
===========


```javascript
is_maybe() // true

is_maybe( false ) // true

is_maybe( 'moin' ) // false

is_maybe( 8 ) // false
```
